# Markdown笔记应用产品需求文档PRD（v1.0）
**最后更新时间：** 2026年7月13日

## 1. 产品概述
本产品是一款轻量级、高效的Markdown笔记应用，旨在为用户提供一个无干扰的文本记录与排版环境。应用采用经典的“列表+编辑+预览”三栏布局，支持Markdown语法的实时渲染，特别强化了代码块的语法高亮功能，使其不仅是普通的文本工具，更是程序员、技术写作者进行知识管理与记录的利器。

## 2. 目标用户
* **程序员/开发者：** 需要频繁记录代码片段、技术方案及API文档，对代码块的排版和语法高亮有硬性需求。
* **技术写作者与博主：** 习惯使用Markdown进行日常写作，需要边写边看最终排版效果。
* **学生与知识管理爱好者：** 使用Markdown整理课堂笔记、读书笔记，追求工具的简洁与高效。

## 3. 核心功能
### 3.1 笔记管理
* **创建笔记：** 用户可以创建新笔记，需输入或自动生成“笔记标题”，并在编辑区输入“Markdown内容”。
* **删除笔记：** 在笔记列表中提供删除按钮（或滑动/右键菜单删除），删除前需有二次确认弹窗，防止误删。
### 3.2 编辑与实时预览
* **双栏联动：** 界面分为左右两部分，左侧为纯文本编辑器，右侧为Markdown实时预览区。
* **实时渲染：** 用户在左侧输入内容时，右侧预览区需实时（防抖处理，如300ms内）更新渲染结果。
* **光标同步（进阶）：** 滚动左侧编辑器时，右侧预览区同步滚动至对应位置（可选）。
### 3.3 Markdown渲染与代码高亮
* **标准Markdown支持：** 支持标题、加粗/斜体、引用、无序/有序列表、超链接、图片、表格等基础语法。
* **代码块语法高亮：** 针对包裹在 ``` 中的代码块，自动识别语言类型（如JavaScript, Python, Java等）并应用不同颜色的语法高亮，支持行号显示，提升专业感。
### 3.4 笔记列表与搜索
* **列表展示：** 在应用最左侧显示所有笔记的列表，按“最后修改时间”倒序排列，列表项显示标题及部分内容摘要。
* **快速搜索：** 列表顶部提供搜索输入框，支持根据“笔记标题”和“正文内容”进行模糊搜索，实时过滤笔记列表。

## 4. 界面设计
### 4.1 整体布局
应用采用**三栏式**宽屏布局（从左至右）：
* **第一栏（左侧边栏，宽度约250px）：** 笔记列表区。
* **第二栏（中间区，自适应宽度）：** Markdown编辑器。
* **第三栏（右侧区，自适应宽度）：** 实时预览区。
### 4.2 详细组件说明
* **左侧边栏：**
  * 顶部：搜索输入框（带搜索图标）。
  * 搜索框下方：「+ 新建笔记」主操作按钮。
  * 主体：笔记列表卡片。每张卡片包含“标题”、“最后更新时间”、“内容摘要（纯文本）”和“删除（垃圾桶）图标”。
* **中间编辑器区：**
  * 顶部：大字号的单行文本输入框，用于输入笔记标题。
  * 主体：多行文本域，支持等宽字体（如Fira Code或Menlo），行高适中，支持Tab键缩进。
* **右侧预览区：**
  * 背景：与编辑器区有轻微的颜色区分，或者通过分割线隔开，以增强层次感。
  * 排版：渲染后的HTML需具备良好的排版样式（合理的行距、段间距、字体大小层级），代码块需带有圆角边框、深色背景和彩色高亮字符。
### 4.3 交互细节
* 首次打开应用时，左侧列表为空，提示“点击新建笔记开始记录”。
* 点击左侧列表中的任意笔记，中间编辑器和右侧预览区同步加载该笔记的数据。
* 预览区内的代码块右下角可提供“复制代码”快捷按钮。
